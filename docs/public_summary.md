# Public Summary

This public summary describes the thesis analysis at a high level without sharing unpublished raw data or complete result files.

## Dataset Summary

- Organism: male Sprague Dawley rats
- Groups: control and hypobaric hypoxia
- Samples analyzed: 20 total samples, 10 per group
- Data type: untargeted LC-MS/MS metabolomics
- Features detected in the raw processed table: 2,433 metabolites/features

## High-Level Findings

The private exploratory analysis suggested that hypobaric hypoxia was associated with changes in the fecal/gastrointestinal metabolomic profile of rats.

Public-safe findings can be described as:

- PCA showed group-level separation in the exploratory analysis.
- A subset of metabolites/features passed screening criteria.
- Major biological themes included amino acid metabolism, lipid metabolism, stress response, and host-microbiome related metabolism.

Exact metabolite names, intensity values, p-values, fold changes, VIP scores, AUC values, and complete result tables are not included in this public repository.

## Interpretation

The findings are hypothesis-generating. Untargeted metabolomics annotations can be uncertain, and candidate biomarkers should be validated using targeted LC-MS/MS, authentic standards, and additional biological replication before making strong mechanistic claims.
