# Results

This folder is intentionally kept free of private thesis results.

When the demo pipeline is run, generated outputs should be written to a separate folder such as `results_demo/`. Those outputs are based on simulated data only.

Private thesis outputs should remain local unless public sharing is approved. Examples of private outputs include:

- filtered data matrices;
- PCA score tables and real PCA figures;
- t-test, FDR, and log2 fold-change tables;
- volcano plots based on thesis data;
- PLS/VIP result tables and figures;
- biomarker tables;
- pathway input tables;
- ROC curves and AUC values for candidate biomarkers.

For a public portfolio, report only high-level summaries such as the number of samples, number of features detected, broad pathway themes, and whether exploratory PCA separation was observed.
